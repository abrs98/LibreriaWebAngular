Para poder ejecutar la aplicacion sera necesaro:

****************** Levantar Backend *******************************
1 Descargar proyecto de avance 4 en Github:   https://github.com/Marxdd/LibreriaWeb.git

2. Crear la base de datos dbbiblioteca en mongoDB y las colecciones usuarios y libros. Todo esto en localhost:27017.

3. Ejectuar en Node.js el archivo backend/app.js



****************** Ejecutar Login En Angular **********************

1 Descargar proyecto de avance final:   https://github.com/abrs98/LibreriaWebAngular.git

2. Ejectuar el comando ng serve dentro la carpeta first-angular-app
